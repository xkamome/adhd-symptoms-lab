# adhd — ADHD 症狀實驗室

一個**可擴充的合集**：把 ADHD 的內在經驗，各自做成可親身體會的小機制／濾鏡，全部放在這一個資料夾。
首頁列出所有機制，點進去玩；可重用的濾鏡放在 `src/shared/`。

調性：純體驗 / 解謎好玩；主題是優雅外衣，但機制本身要讓人「感覺到」那個症狀。

## 結構
```
src/
├─ App.svelte            # 首頁選單（registry 驅動）+ 返回
├─ mechanics/

└─ assets/painting.jpg   # 布勒哲爾《Netherlandish Proverbs》(公共版權, Wikimedia)
```

## 機制一：超專注隧道 (Hyperfocus)

**經驗**：超專注時你看得無比仔細，但視野縮成一條隧道，錯過周邊發生的一切。

**玩法**：
- 畫面是一幅**模糊**的名畫。游標／手指**停住不動**，視野會慢慢**縮窄**、中心**解析度越來越高**；一移動就重置。
- 任務：找出藏在畫裡的 **3 隻貓**（貓是疊在名畫上的 sprite，會一起被模糊，對焦到才看得清）。
- **找到 = 對焦夠清晰時，把中心停在貓上約 0.35 秒**（自動認定，不用點選）。
- **超專注的代價**：你在對焦時，周邊會偶爾閃過小光點；因為你盯著中心、周邊又模糊，很容易**錯過**。結算會告訴你錯過了幾件。
- 找齊 3 隻貓過關，顯示「你錯過了 N 件周邊的事」。

**可重用濾鏡 `FocusTunnel`**：輸入一張圖（可選 `decorate` 把疊加物烘進每層一起模糊），提供
`resize / update(dt, pointer) / render(ctx)`、`focus` 狀態與 `imgToScreen()` 座標轉換。
之後別的機制（或別的 prototype）可直接拿去用。

## 範圍
### v0（本次）
- FocusTunnel 濾鏡 + 一幅名畫 + 3 隻貓 + 停住對焦/找到 + 周邊錯過機制 + 過關結算
- 首頁選單（可擴充）

### v1（之後，加在同資料夾）
- 更多 ADHD 症狀機制：無法啟動、不在眼前就消失、時間盲…
- 每關不同名畫/目標、難度、音效、對焦音與呼吸感

## 技術
Svelte 5 + TS。FocusTunnel 用 Canvas 2D（離屏預算多層模糊 + 徑向遮罩柔邊隧道），全程裝置像素運算。
